# RGB_Sensing
There is an uncountable number of colors in the world. Mixing colors will give birth to new colors. For the benefit of studying, people identified three primary colors that give birth to all visual colors. They are Red, Green & Blue. RGB Color Sensor is a product that can identify the colors it detects from a surface. Our task was to build an RGB Color Sensor from the basics, neglecting the condition that there is a color sensing module. So we touch on the principles of behavior of the light. Through these fundamentals, we succeeded in creating an RGB Color Sensor using basic electronic components like LEDs and LDRs. And use some data processing methods to increase the accuracy of the color sensing mechanism.
